Quaternion Function Reference

mpower

^ Matrix power
(Quaternion and octonion overloadings of standard MATLAB® function)

Syntax

Z = X ^ Y

Description

X ^ Y raises the quaternion matrix X to the power Y.

If both parameters are scalar, the .^ function is called. Otherwise, for real integer Y (including negative integers), quaternion matrices are raised to the given power. This is done by repeated squaring. For non-integer Y, a matrix logarithm/exponential formula is used. CAUTION, this may not give correct results in all cases.

In the octonion case, the power of a matrix is ambiguous for Y > 2 (octonion matrix multiplication is not power associative, unlike octonion multiplication). Hence only integer powers of 0, 1 and 2 are implemented.
