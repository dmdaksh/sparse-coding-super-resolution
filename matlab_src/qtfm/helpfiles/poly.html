Quaternion Function Reference

poly

Characteristic polynomial of a Hermitian quaternion matrix
(Quaternion overloading of standard MATLAB® function)

Syntax

P = poly(A)

Description

poly(A) computes the characteristic polynomial of a quaternion matrix (with limitation to Hermitian matrices).

For non-Hermitian quaternion matrices, use the adjoint function to construct a complex matrix, and then use the MATLAB® poly function to compute the characteristic polynomial of the adjoint matrix. This characteristic polynomial will be of order one greater than the dimension of the quaternion matrix, because the adjoint matrix is double the dimension of the quaternion matrix.

Examples

>> q = randq(3)

q = 3x3 quaternion array
>> Q = q * q'

Q = 3x3 quaternion array  % Make a Hermitian quaternion matrix.
>> P = poly(Q)

P = 1.0000   -9.0000   17.1475   -7.0271

>> roots(P)

ans =

    6.5436
    1.8874
    0.5690

>> eig(Q)

ans =

    6.5436
    0.5690
    1.8874

% Notice that the roots of the characteristic polynomial of Q are the same
% as the eigenvalues.
